Welcome to Dummy SMTP Server

This is a Dummy SMTP Server for Developers, written in C#.

This application is under heavy development

This application will not be able to send or receive real emails, but rather can be used as a dummy server to test your email sending applications which is running in the local host.

Getting started

Getting started

Stable release (recommended)

  1. Download the latest binary zip file from this link Dummy.SmtpServer.binary.zip
  2. Extract the zip and execute the file.

Development release

Clone the master repo

git clone https://github.com/farajfarook/dummy-smtp-server.git

Open the solution with Visual Studio and build it!
